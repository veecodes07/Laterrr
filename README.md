# 🐊 Laterrr

> *later, alligator.*

A lightweight Chrome extension to save anything you find on the internet — and actually come back to it.

---

### why
bookmarks exist but nobody uses them.  
you find something cool on instagram, search it up, forget it by tomorrow.  
Laterrr fixes that.

---

### what you can save

| tag | for |
|-----|-----|
| 🛍 Shop | that fit you'll buy after salary |
| ▶ Watch | the youtube rabbit hole for later |
| 🎮 Game | steam sale finds |
| ✅ Task | forms, deadlines, adult stuff |
| ✨ Inspo | things that made you feel something |
| 📖 Read | articles you'll *definitely* read |

---

### features
- one click save from any tab
- **right-click any link** on any page → "🐊 Save to Laterrr" — no popup needed
- deadline alerts for tasks — chrome notifies you on the day
- filter by category
- notes on every save
- export all your saves to CSV
- syncs across your chrome devices

---

### install

```
1. clone this repo
2. go to chrome://extensions
3. enable Developer Mode (top right)
4. click Load Unpacked → select the Laterrr folder
5. pin it. use it. remember things.
```
